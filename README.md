# Swarm Enterprise OS v1.0
**The World's First Self-Replicating Autonomous Digital Factory**

## 1. Overview
Swarm Enterprise is a "Programmable Company in a Box." It provides an elite, 8-agent autonomous software engineering workforce that lives on your private hardware. It handles the entire SDLC (Software Development Life Cycle) from Vibe-to-Production with zero recurring API costs.

## 2. Infrastructure Requirements
- **Host:** Windows 10/11 or Windows Server 2025.
- **Runtime:** Docker Engine (Linux Mode) via WSL2.
- **Brain:** Ollama running on a local machine (CPU/RAM optimized).
- **Ingress:** Cloudflare Zero Trust Tunnel.

## 3. Quick Start
1. Edit the \`.env\` file with your unique keys (Stripe, Cloudflare, Groq).
2. Run \`./boot_swarm.ps1\` from an Administrator PowerShell.
3. Access your dashboard at your registered domain.

## 4. The Fractal Hierarchy
- **Level 1 (The Board):** 10 Managers performing strategic decomposition.
- **Level 2 (Supervisors):** 30 Department Leads managing task atomization.
- **Level 3 (Workers):** Adversarial pairs (Executor/Critic) performing physical file writes.
