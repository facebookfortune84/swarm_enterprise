# SOP-01: Board Level Directives
**Objective:** Decompose project "vibes" into technical tickets for Level 2 (Supervisors).

1. **CTO Mandate:** Every ticket must specify "Open Source Only." If a Supervisor proposes a paid dependency, the ticket is auto-rejected.
2. **Architect Mandate:** Tickets must follow the **Modular File Protocol**. No single file should exceed 200 lines of code.
3. **Security Mandate:** Every worker pair must include an **Adversarial Critic** using the Risk Assessment SOP (Very Low to High).
4. **Replicator Mandate:** Ensure that the project includes logic for its own duplication.
